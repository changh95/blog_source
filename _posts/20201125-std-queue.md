## std::queue

- FIFO (First In First Out) 구조
  - `std::queue`에서의 `push()`는 `std::stack`에서의 `push_back()`을 의미
  - `std::queue`에서의 `pop()`는 `std::stack`에서의 `pop_front()`를 의미
- 기본적으로 `std::deque`를 사용해서 만든 래퍼 클래스.
  - 모든 함수가 $O(1)$으로 사용 가능. 

## 사용 예제

```cpp
std::queue<int> q;
q.push(1);
q.push(2);
q.push(3); // {1,2,3}

q.pop(); // {2,3}
q.push(4); // {2,3,4}



```
